◆作品概要
◇使用言語	: C++

※注意※プレイには2本のアナログスティックを持つコントローラーが必須です。


◆作品説明
妖怪たちを切り捨てろ！直感的で爽快な斬撃アクション！
[直感的な操作][爽快なリアクション]の２点をコンセプトとして制作しました。
「LIBLADE」というアクションゲームをリスペクトしています。
制作期間が長く一部仕様が迷走、仕様変更を繰り返して現在の形になっています。

◆工夫した点
◇攻撃の当たり判定
	[直感的な操作]の実現のため360°自由な攻撃を実装。
	斜めに攻撃をした際に見た目通りの判定を取るため線分と円の当たり判定を作成しました。
	
◇エフェクトのカプセル化
	[爽快なリアクション]の実現のためテクスチャアニメーションのエフェクトを実装。
	テクスチャの差し替えなどで試遊をする中で簡単にエフェクトを生成できるクラスを作成しました。
	その際、他のプロジェクトなどでも汎用的に使えるようシンプルなクラスを意識しました。

◇自作モデルの使用
	個人制作では初めての人型キャラクターを実装。
	モデリングからモーション作成まで全て自分で行ったこだわりのプレイヤーです。

◆反省点
◇モチーフにしたゲームを模倣しようとするあまり仕様を詰め込みすぎた。
	↳身の程を弁えた仕様にすることを学んだ。夢を見て目標を高く持ちすぎると痛い目を見る。

◇ゲームの面白さにこだわりプログラムが蔑ろに
	↳演出や難しい当たり判定に時間をかけすぎてしまったため
	今後の制作ではスケジューリングを意識することを決意。

◆操作方法
[Xboxコントローラ]
左スティック	: 移動
右スティック	: 傾けた方向に攻撃
LBボタン		: ジャンプ
RBボタン		: 押している間攻撃がダッシュに変化
Aボタン			: 画面遷移
