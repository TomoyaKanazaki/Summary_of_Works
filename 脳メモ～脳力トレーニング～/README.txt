Windows11

◆作品説明
この作品はJEMTCの主催する脳トレゲーム制作コンペティションでグランプリを獲得した作品です。
学校とは関係ないプライベートでの参加だったので普段の授業ではチームになることがない先輩に声をかけて制作しました。
作品の内容としては脳トレをテーマに記憶力と判断力を問う４種類のミニゲームを収録したミニゲーム集となっています。

◆担当箇所
◇ミニゲーム「でたじゅんばんにおして」
◇ハイスコアの外部ファイル入出力処理

◆工夫した点
◇ミニゲームのレベルデザインを効率化するため外部ファイルを活用しました。
◇デバッグ/リリースビルドの際に読み込む外部ファイルを別にすることでデータの改ざんを防止しました。

◆反省点
◇先輩の基盤を使用したこともあり、多くの点で先輩に頼り切りになってしまった。
	↳他人のコードをしっかりと読んで理解することを意識するようになりました。
	
◇授業での制作と並行して制作を行ったためメリハリをつけることに失敗した。
	↳Excelを活用しタスクの可視化をすることでスケジュール管理を徹底するようになりました。

◆操作方法
◇マウスカーソルで左クリック
◇タブレットPCのタッチパネル対応
